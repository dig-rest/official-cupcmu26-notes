# official-cupcmu26-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，便于查阅、分享和长期维护。仓库不针对任何特定域名或网站，仅作为通用页面存档使用。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的样式、脚本、图片等资源
├── README.md       # 本说明文件
└── index.html      （可选）仓库首页或导航页
```

- **pages/**：每个 HTML 页面以独立文件形式存放，文件名应反映页面内容或用途。
- **assets/**：若页面共用资源（如 CSS、JS、图片），建议统一放置于此目录。
- **index.html**：可放置一个简单的导航或索引页面，便于快速浏览所有归档页面。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面内容应保持原样归档，如需更新，请直接修改对应文件并提交。
- 归档范围包括但不限于：笔记、文档、工具界面、信息展示等通用用途。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面归档或修正现有页面。
- 提交新页面时，请确保文件结构清晰，资源引用路径正确。
- 不建议在页面内添加追踪脚本、广告或外部依赖，以保持归档的纯净性与可移植性。
- 如发现页面失效或内容错误，可提交 Issue 说明情况。
